# 探索報告 {#discover-reporting}

在[!UICONTROL User Settings]內，使用者可以選擇將貨幣從預設貨幣變更為任何列出的公司貨幣。 這會變更金額（例如，成本、收入及Pipeline收入）的顯示方式，連同三個字母的貨幣代碼。
