# Discover報表 {#discover-reporting}

內 [!UICONTROL User Settings]，使用者可以選擇將貨幣從預設貨幣變更為列出的任何公司貨幣。 這將更改「成本」、「收入」和「管道收入」等金額的顯示，以及三字母貨幣代碼。
