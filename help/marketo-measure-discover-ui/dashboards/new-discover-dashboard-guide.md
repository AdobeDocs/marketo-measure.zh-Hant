* **日期型別、量度、Dimension、分組依據**：這些設定現在已為每個儀表板預先定義，以便最佳化深入分析傳遞。 如需詳細資訊，請參閱工具提示和隨附檔案。
* **行銷活動Source**
* **帳戶ID和名稱**：正在檢閱。
