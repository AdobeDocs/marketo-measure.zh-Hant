# 收入總覽儀表板 {#revenue-overview-dashboard}

收入概觀儀表板提供從CRM衍生的總收入的深入分析，讓您更清楚瞭解行銷策略的角色。 它概略說明行銷如何影響您的整體收入，以及如何協助成功完成交易。

展示板回答的問題：

* 我們的行銷活動佔總收入的百分比為何？
* 我們的行銷活動影響了多少百分比的「成功交易」？
