## 收入圖磚 {#revenue-tile}

![](assets/one.png)

根據歸因模型，來自Closed Won商機的歸因收入總計。

 — 機會詳細資訊：顯示相關的Opportunity ID 、 Created Date 、 Close Date 、 ISO Code和Revenue Attribution的劃分。\
 — 顯示詳細資訊：按月劃分。

## 概述（圖表）圖磚 {#overview-chart-tile}

![](assets/two.png)

圖表會顯示各管道的收入，並隨時間顯示。

 — 機會詳細資訊：顯示相關的Opportunity ID 、 Created Date 、 Close Date 、 ISO Code和Revenue Attribution的劃分。\
 — 顯示貢獻Op ID的劃分：

* 按ROI周
* 按ROI資料
* 按ROI小時
* 按ROI分鐘
* 按ROI時間

## 支出圖磚 {#spend-tile}

![](assets/three.png)

指定時段內來自API成本和自我報告成本的總支出。

>[!NOTE]
>
>鑽進不到一個月， [!DNL Marketo Measure] 會平均該月的花費，並將該平均值分配至每天。

 — 支出詳細資訊：顯示相關「支出促銷活動唯一ID」、「促銷活動名稱」、「管道」、「子管道」、「日期」、「原始貨幣」、「支出」的劃分。   — 顯示詳細資訊：按月劃分。

## ROI圖磚 {#roi-tile}

![](assets/four.png)

總收入（來自所選歸因模型）和總支出(收入圖磚÷支出圖磚= ROI圖磚)中計算出的ROI（投資報酬率）。

## 交易標題 {#deals-tile}

![](assets/five.png)

顯示歸因交易總數（「交易」是「已結韓元」機會的計數）。

>[!NOTE]
>
>如果選取「接觸點日期」日期類型，可能會有部分計數，因為某些接觸點可能不在時間範圍內。

 — 機會詳細資訊：顯示相關Opportunity ID 、 Created Date 、 Close Date和Attribution(count)的劃分。   — 顯示詳細資訊：按月劃分。

## 每筆交易成本圖磚 {#cost-per-deal-tile}

![](assets/six.png)

獲得一個成功機會所需的平均成本。 總支出除以交易總數(「支出磁貼」 ÷「交易磁貼」=「每筆交易成本」)。

## 管道收入圖磚 {#pipeline-revenue-tile}

![](assets/seven.png)

未結Opportunity的潛在收入總額

>[!NOTE]
>
>收入會反映在指定的歸因模型中。

 — 機會詳細資訊：顯示相關的Opportunity ID 、 Created Date 、 Close Date 、 ISO Code和Revenue Attribution的劃分。   — 顯示詳細資訊：按月劃分。

## 交易規模圖磚 {#deal-size-tile}

![](assets/eight.png)

Closed Won商機的平均收入(Revenue Tile ÷ Deals Tile = Deal Size Tile)。

## 通道摘要圖磚 {#channel-summary-tile}

![](assets/nine.png)

根據所選量度（收入、管道收入、支出、交易、機會、銷售機會、聯繫人）顯示管道清單。

## 子頻道摘要圖磚 {#subchannel-summary-tile}

![](assets/ten.png)

根據所選量度（收入、管道收入、支出、交易、商機、銷售機會、聯繫人）顯示子管道清單。

## 促銷活動摘要圖磚 {#campaign-summary-tile}

![](assets/eleven.png)

根據所選量度（收入、管道收入、支出、交易、商機、銷售機會、聯絡人）顯示促銷活動清單。

## ROI（圖表）拼貼 {#roi-chart-tile}

![](assets/twelve.png)

根據選取的歸因模型，顯示歸因於所選量度（管道、子管道、促銷活動、帳戶、廣告群組、廣告、廣告商、創意、關鍵字、版位、網站）的ROI。

## ROI摘要圖磚 {#roi-summary-tile}

![](assets/thirteen.png)

根據選取的歸因模型，顯示歸因於所選量度（管道、子管道、促銷活動、帳戶、廣告群組、廣告、廣告商、創意、關鍵字、版位、網站）的ROI。

 — 機會詳細資訊：顯示相關Opportunity ID 、 Created Date 、 Close Date和Attribution(count)的劃分。

 — 顯示詳細資訊：按月劃分。
